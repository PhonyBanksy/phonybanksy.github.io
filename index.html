How to Use

  1. In MotorTown: Open your event and export the route.
  2. On this site: Use Ctrl+V to paste into the Input window.
  3. Configure:
    • Check "Reverse?" to swap start/finish.
    • Check "Boxes?" to make gate depth equal to width.
  4. Resize: Use the dropdown to enlarge gates.
  5. Interact: Click points on the map to Adjust dimensions or Rotate.
  6. Finalize: Press Process, then Copy Output.
  7. Import: Import the code back into the event in game.

Clearing browser cookies/cache will also clear all saved routes on this website.
