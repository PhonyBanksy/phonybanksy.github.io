How to Use

  1. In MotorTown: Open your event and export the route.
  2. On this site: Use Ctrl+V to paste into the Input window.
  3. Configure: Check/uncheck "Reverse?" to swap start with finish.
  4. Resize: Use the dropdown to enlarge gates if they are too small for large events.
  5. Resize: Click on individual gate to change only THAT gate width.
  6. Finalize: Press Process, then Copy Output.
  7. Import: Import the code back into the event in game and enjoy!

Clearing browser cookies/cache will also clear all saved routes on this website.
